API Documentation

Welcome to the Mizofy REST API. This API allows you to resolve TeraBox shared links, stream files with range seek support, and manage account session cookies. The API is cross-origin resource sharing (CORS) compatible, meaning you can easily connect your Hostinger websites, custom landing pages, or mobile apps directly to this backend!

GET /api/resolve
Resolves a shared TeraBox link. Iterates through the active session cookies pool automatically, bypasses captcha/rate-limit blocks, masks streams, and returns a structured list of files.
Query Parameters
Parameter Type Required Description
url string Yes The shared TeraBox URL (e.g. 1024terabox.com/s/...). Supports all domain aliases.
ndus string No Optional manual ndus cookie to bypass the database cookie pool.
Example Response
{
  "errno": 0,
  "list": [
    {
      "server_filename": "movie.mp4",
      "size": "154329291",
      "category": "1",
      "duration": "3600",
      "dlink": "/api/download?id=fa15f88b76934f5ba0ebf88e57fd7750",
      "thumbs": {
        "url3": "/api/thumbnail?id=836651b2b8cc4e7b951cd3c7b4499b8f"
      }
    }
  ]
}
GET /api/download
Streams or downloads the binary file content. Supports HTTP Range requests (status 206) for seekable video playback.
Query Parameters
Parameter Type Required Description
id string Yes The secure UUID token generated during link resolution.
mode string No Set to attachment to force a browser download (Content-Disposition).
Supported Headers
Header Value Description
Range bytes=X-Y Required for seekable video player controls. App forwards range headers to TeraBox CDN.
GET /api/thumbnail
Fetches and proxies thumbnail images from TeraBox servers. Ensures your visitors can see video previews without exposing your active session keys in raw image source logs.
Query Parameters
Parameter Type Required Description
id string Yes The secure thumbnail token.
GET /api/admin/cookies
Protected with Basic Authentication (admin123 / suman821310). Retrieves all session items, active rate-limit status, and configurations from the database.

*Supports POST (with action: "add" / "update" / "delete") and DELETE (to purge pool).
CORS Integration Example (Hostinger / Javascript)
const API_BASE = "https://your-koyeb-backend.koyeb.app";
const headers = {
  "Authorization": "Basic " + btoa("admin123:suman821310"),
  "Content-Type": "application/json"
};

const r = await fetch(`${API_BASE}/api/admin/cookies`, { headers });
const data = await r.json();
console.log("Cookie Pool:", data.pool);
